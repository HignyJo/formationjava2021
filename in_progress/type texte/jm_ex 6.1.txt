6.1	Recherche caractère
-----------------------

Ecrire le programme qui demande à l'utilisateur d'entrer un texte et un caractère.

Le programme affichera :

	La première position où le caractère se trouve dans le texte source
	La dernière position
	Le nombre de fois qu'il apparait
	
Si le caractère ne se trouve pas dans le texte, le programme affichera un simple message

------------------------------------------------------------------------------------------------

Solution
--------

debut recherchecaratere

	variables locales
		var premPos	:N <- 0
		var dernPos	:N <- 0
		var cpt		:N <- 0
		var lgText	:N <- 0
		var texte	:T <- " "
		var car		:T <- ' '
		var nbCar	:N <- 0
	fin variables locales
	
	afficher "Saisissez un texte"
	saisir texte
	
	afficher "Saisissez un caractère"
	saisir caractère
	
	lgText <- longueur (texte)
	
	tant que (cpt < lgText)
		si (car = caract(texte, cpt))
			alors
				nbCar <- nbCar + 1
				cpt <- cpt + 1
		fin si
	fin tant que
	
	afficher "Première position où le caractère apparait : " premPos
	afficher "Dernière position où le caractère apparait : " dernPos
	afficher "Le nombre apparait " + nbCar + " fois"

fin recherchecaratere
